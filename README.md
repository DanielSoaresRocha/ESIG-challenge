## Desenvolvimento 🎬

Clonando o repositório:

```
git clone https://github.com/DanielSoaresRocha/ESIG-challenge.git
```

Navegando até a pasta do repositório:

```
cd Esig/src
```

Baixando as dependências

```
npm i
```

Rodando o projeto

- Abra o arquivo `index.html` no seu navegador
